# ASCII Mockups

This document outlines the visual structure of the core pages for the Muhassaba app based on the UX decisions made on 2026-06-11.
(The app UI itself is purely Arabic).

## 1. Day Screen Layout (اليوم)
Primary view for filling out the daily scorecard. Uses a flat list. Completed tasks move to the bottom.

```text
+------------------------------------+
| ☰                الدرجة: ٨/١٠      |
|  <  السبت  الأحد  الإثنين  الثلاثاء> |
|      ٧٠٪   ٥٠٪    ٨٥٪    --        |
+------------------------------------+
| ▼ الصبح                            |
| [ ] الجماعة الأولى                 |
| [ ] أذكار الصباح             [ 📖 ] |
+------------------------------------+
| ▼ الظهر                            |
| [ ] الجماعة الأولى                 |
| [ ] السنن الرواتب                  |
+------------------------------------+
| ▼ القرآن                           |
| [ ] قراءة جزء من القرآن    (١٢/٢٠) |
|  (تتوسع لتعرض شريط تمرير عند الضغط)|
+------------------------------------+
|                                    |
| ▶ تم إنجازها (٤)                   |
+------------------------------------+
| [ اليوم ]   [ الأسبوع ]   [ الأذكار ] |
+------------------------------------+
```

## 2. Week Screen Layout (الأسبوع)
Appears only if the user has weekly tasks configured. Functions similarly to the Day Screen but for weekly goals.

```text
+------------------------------------+
| ☰                الأسبوع ٤٠٪       |
|      <  ٨ يونيو - ١٤ يونيو  >      |
+------------------------------------+
| ▼ أعمال الأسبوع                    |
| [ ] زيارة القبور                   |
| [ ] سنن الفطرة                     |
+------------------------------------+
|                                    |
| ▶ تم إنجازها (١)                   |
+------------------------------------+
| [ اليوم ]   [ الأسبوع ]   [ الأذكار ] |
+------------------------------------+
```

## 3. Dhikr Reader
Opened by tapping on a dhikr deed in the Day Screen. Uses a vertical scrollable list with individual counters.

```text
+------------------------------------+
| < رجوع         أذكار الصباح        |
+------------------------------------+
|                                    |
|  آية الكرسي                        |
|  اللَّهُ لَا إِلَهَ إِلَّا هُوَ الْحَيُّ...      |
|                                    |
|         ( اضغط للعد )              |
|            [ ١ / ١ ]               |
|                                    |
+------------------------------------+
|  سورة الإخلاص                      |
|  قُلْ هُوَ اللَّهُ أَحَدٌ...               |
|                                    |
|            [ ٣ / ٣ ]               |
+------------------------------------+
```

## 4. Mutlaq Dhikr Counter Page (الأذكار المطلقة)
Accessed via the bottom tab. Features a split view: select a dhikr from the top list, and use the persistent keypad at the bottom to quickly log amounts.

```text
+------------------------------------+
| ☰                الأذكار المطلقة   |
+------------------------------------+
| ► استغفار          (محدد)          |
|   الحالي: ٢٠٠                      |
+------------------------------------+
|   الصلاة على النبي                 |
|   الحالي: ٥٠                       |
+------------------------------------+
|   سبحان الله                       |
|   الحالي: ٠                        |
+------------------------------------+
|                      ( + ) جديد    |
+====================================+
|                                    |
|   [ +١ ]   [ +٥ ]   [ +١٠]         |
|   [ +٢٥]   [ +٥٠]   [+١٠٠]         |
|                                    |
|   [ مخصص: ___ ]     [ حفظ ]        |
+------------------------------------+
| [ اليوم ]   [ الأسبوع ]   [ الأذكار ] |
+------------------------------------+
```

## 5. Edit Scorecard (تعديل الجدول)
Accessed from the Settings menu in the Drawer. Used to customize the daily template.

```text
+------------------------------------+
| < رجوع         تعديل الجدول        |
+------------------------------------+
| أعمال الأسبوع                      |
| = زيارة القبور                [🗑] |
| = سنن الفطرة                  [🗑] |
|   + إضافة عمل أسبوعي               |
+------------------------------------+
| الصبح                              |
| = الجماعة الأولى              [🗑] |
| = أذكار الصباح                [🗑] |
|   + إضافة عمل إلى الصبح            |
+------------------------------------+
| الظهر                              |
| = الجماعة الأولى              [🗑] |
|   + إضافة عمل إلى الظهر            |
+------------------------------------+
|                                    |
|   + إضافة قسم جديد                 |
+------------------------------------+
```

### Library Picker Modal
Triggered when tapping `+ Add Deed`. Shows curated predefined tasks to choose from, or an option to create a custom one.

```text
+------------------------------------+
|                                    |
|           اختر عملاً               |
|                                    |
| [🔍 ابحث في المكتبة...           ] |
|                                    |
| أعمال مقترحة                       |
| + صلاة الضحى               (يومي)  |
| + قراءة سورة الملك         (ليلي)  |
| + أذكار المساء             (ذكر)   |
| + صيام الاثنين والخميس    (إثن/خمي)|
|                                    |
| + [ إنشاء عمل مخصص ]               |
|                                    |
|                      [ إلغاء ]     |
+------------------------------------+
```

### Deed Settings Modal
Triggered when tapping a specific deed in the Edit Scorecard view to configure its schedule and weight.

```text
+------------------------------------+
|                                    |
|      تعديل إعدادات العمل           |
|                                    |
| الاسم: [ صيام الاثنين والخميس    ] |
| النوع: [ ثنائي ▼ ]                 |
| القسم: [ الصبح ▼ ]                 |
|                                    |
| نوع التكرار:                       |
| ( ) يومي                           |
| (•) أيام محددة                     |
|     [ ] س [•] ح [ ] ن [ ] ث        |
|     [•] ر [ ] خ [ ] ج              |
| ( ) أسبوعي (في أي وقت)             |
|                                    |
|           [ حذف ]       [ حفظ ]    |
|                                    |
+------------------------------------+
```

```text

+------------------------------------+
|      التقدم الأسبوعي (رسم بياني)   |

|  /    \/\/  \                      |

| الصبح      |   |   |   |   |   |   |

|        < تمرير أفقي >              |
+------------------------------------+
```
